Basic faker.js Demo





Steps


  1. Insert CDN <link> to use faker
  2. Insert <script src="script.js"> and create external file
  3. Add event listener for button
  4. Store faker.helpers.createCard(); to info variable to be able to access API methods
  5. Use destructuring to store faker data into a variable that should match faker API varnames
  6. Use the destructured variables to grab data and render using the DOM
